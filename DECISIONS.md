# KWCI 결정 로그 (DECISIONS) — 사고 과정 인계

> "왜 이렇게 만들었는가"의 기록. 외부 검토자(Codex 등)가 설계 의도를 이해하고 이어받기 위한 문서.

## 주요 결정과 근거

1. **8 도메인 체계 채택(구 6장르 폐지)**
   - 근거: 산업통계 가용성·플랫폼 측정 가능성·소비 양식·정책 단위 정합. 영상/웹툰/뷰티/패션/관광을 분리.
   - 폐기: K-drama/K-movie/K-culture식 6장르 구분(대시보드 잔재였고 통계 단위와 불일치) → 전면 제거.

2. **3 레이어 가중 0.5/0.3/0.2 + 결측 재정규화**
   - 근거: 실물 경제(L1)를 닻으로, 플랫폼(L2)·수용자(L3)가 단기 변동 보정. 결측 레이어는 남은 층 가중을 재정규화.

3. **정규화 이중 체제**
   - 시계열=2018=100 지수화(성장 추세), 횡단=0~100 단면(국가 상대비교). 목적이 다른 두 산출물이라 분리.
   - 두 값 직접 비교 금지. (이전 판에서 이 구분이 모호했던 것을 명시화)

4. **관광(ktourism) L1 소스 진화**
   - KTO 월별 REST(15000297)는 2018년 미제공 → 초기엔 공식 연간통계 임베드 → 최종 **KOSIS 국적·지역별 외국인 입국자(orgId=111, DT_091_111_2009_S005A)** 로 라이브화(2018 포함).
   - 2025는 표 미수록 → 공표 연간(약 1,870만, 지수 122)으로 보강. 2019 국적별 분해는 표에 결측 → 보간.
   - 검증: KOSIS 총계 2018 1,563만 → 2024 1,697만(2018=100→108.6), 기존 추정(108.9)과 일치.

5. **K웹툰 L1 = KOSIS '만화 수출'**
   - 콘텐츠산업조사 만화산업=출판만화+웹툰. 2020년부터 웹툰이 매출 과반·수출 주도. 2018 저베이스로 증가 배수 과대 보임에 유의.
   - 산업 '매출'(2024 2.29조, 4.9배)은 내수 포함이라 별도 해석.

6. **K푸드 L1 = 가공식품 HS(라면·김치·소스·조미김 등)**, 농식품 총수출(곡물·축산 포함) 아님 → 한류 식품 신호에 초점.

7. **시장 다변화 지수(별도 항목, 종합지수 미편입)**
   - ENM = 1/HHI, HHI = Sum(share^2). 볼륨과 분리한 '질·복원력' 지표.
   - 관광 source-market(국적 구성, 2018=100): 코로나기 분산 최고(2022≈188.6) → 2024 중국 27% 재집중으로 93.8(2018 아래).
   - 수용 audience(분야별 국가 구성): **1차 기준=KOFICE 설문**. 이유: YouTube 단면은 'API의 지역 인기영상 × 글로벌 조회수' 산물이라 특정국 비중 과장(예: K-pop 말레이시아 25% 아티팩트). 설문 기준에서 K-pop ENM 6.8→14.6, 최다 태국 8.5%로 안정.

8. **Reddit → Google Trends 교체**
   - 근거: 수집 신뢰성·지속성·국가 커버리지. L3 하위가중 KOFICE 0.40/YouTube 0.35/Trends 0.25.

9. **가중 프로파일 4종, 활성=문화중심(alpha=0.4)**
   - industry/equal/two_axis_economic(0.6)/two_axis_cultural(0.4). 프로파일 토글로 강건성(순위 안정성) 점검.

10. **자동화 트리거**
   - on:push(kwci_pipeline/**) + 월 cron + 수동. 과거 'push로 코드만 올리면 파이프라인이 안 돌던' 문제를 on:push 추가로 해결.
   - 키 결측/실패 시 검증고정값(verified-fixed)으로 결정론적 폴백(sources 라벨로 정직 표기).

## Known limitations (검토 시 고려)
- 콘텐츠 4분야(kpop·kvideo·kgame·kwebtoon)의 **국가별 L1 부재**: KOSIS가 전국 단위라 횡단 DSI엔 미반영(2018=100 시계열만).
- **플랫폼 편향**: 중국·러시아는 YouTube/Trends 차단·저점유 → 해당국 과소평가 가능.
- **위험도(R) 미반영**: 반한류·피로도·정치갈등 등 위험 차원 부재.
- 일부 지표 **verified-fixed**(API 장애 시). 절대값보다 구조·순위·추세로 해석.
- **단일 Min-Max 정규화**는 극단값 민감 → 백분위·Z-score 민감도 분석 권장.
- audience 다변화 2019 보간·2025 공표 보강 등 결측 처리 가정 존재.
